== Description ==

Flat Preloader helps you create the ploading page with many excited gif icons.

You can select where the preloading page will be showed, such as home page or all pages.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Flat Preloader screen to configure the plugin
4. Choose your style and enjoy
